Как организованы
онлайн-занятия?

( 1 )

Наш педагог позаботится о том, чтобы ученику было комфортно
на индивидуальных или групповых онлайн-занятии для успешного освоения программы.

( 2 )

Преподаватель в течение
30–40 минут занимается с ребёнком.

( 3 )

Мы предоставляем вам интересные задания, которые можно выполнять самостоятельно между занятиями.

( 4 )

После каждого блока педагог предоставляет родителям обратную связь в формате чек-листа достижений.
